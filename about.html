Change Filter Mode

Toggle between Cuisine and Sentiment Filters. Cuisine filters allow for filtering by restaurant type, while sentiment filters allow for filtering by the aggregated Google Review sentiment analysis rating.

Toggle Buffer Mode

Enable a buffer-click mode to generate a 5km buffer upon click, and show any available restaurant names in the area. Click the button again to return to the regular point and click mode.

Details

Click on any data point to view extra details. Each point displays the restaurant name, sentiment ratings for Food, Service and Atmosphere, Price Index rating from 0-4 indicating user reported expensiveness and Cuisine.
